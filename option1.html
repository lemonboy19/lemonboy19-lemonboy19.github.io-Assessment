WWW Questions and Related


Intro to WWW

Question A



- The world wide web is a platform in which various

interconnected webpages can be accessed using a web browser

or similar application.



- The Web, also known as a client-server system, works by the

client making a request for a particular page

and then the web server should ideally relay back that page to the

client and any other multimedia using HTTP.



- A web server is designed to display website content by

storing, processing, or delivering webpages back to a client.

Web servers implement a version of HTTP and another protocol.



- A web browser is a software application that allows for

retrieving, uploading, or browsing content on the WWW. Each

web page has its own URL which allows for the browser to

identify the resource and retrieve it from the web server.




WWW vs Internet

Question B



- The World Wide Web is the collection of information such

as various webpages that can be accessed via the internet. The internet can

be defined as the network that connects a very large number of computers

and similar devices globally. When comparing the two, the internet can be

thought of as global network of infrastructure powering all the content

and allowing for a variety of interactions between the client and the server.



What is an Intranet

Question C



- An intranet is a private network often found within an enterprise of sorts,

that allows for the secure distribution of company information and

resources between employees and staff. In a sense it’s similar to

the internet but it’s only accessible to authorized people. As mentioned before,

internet refers to the systems that connect various networks of computers

together simultaneously. The internet is public and has users globally.

Conversely, intranet is a type of network that is used privately and has a limited

number of users and information.



Internet Protocols

Question D



- An internet protocol is a set of rules for routing and processing various packets of data

so that they can travel across networks within the infrastructure of the Internet.

As mentioned before, the data that is transported by the internet are called packets.

These rules fully dictate the delivery of data over the internet by managing flow,

access control, etc. There are various types of internet protocols, and some of these

are HTTP, TCP and FTP
